# 海に負けるな!クラゲクラゲちゃん

## 製作期間

December 2025 (approximately 3 weeks)  

## 概要

### 目的・背景

pythonを約８ヵ月勉強したのでその定着確認のために製作しようと思いました。
冬休みに京都水族館に行って、クラゲをたくさん見ました。そこで水槽の中で自由気ままに浮かんでいる
クラゲからインスピレーションを受けてこのゲームを製作しようと思いました。

### 説明

言語はpythonを使いました。pygameを使いゲームを製作しました。
このゲームはクラゲちゃんの目標である空を見に行くというゲームです。
このゲームは海の中をモチーフにしているので何もしていないと自由落下していくようになっています。
操作方法はスペースを押して、マウスを行きたい方向にもっていくだけです。
スペースを長押してパワーをためて、スペースを放すとマウスのある方向に向かってダッシュしていきます。
wasdや矢印ボタンで動かさないようにすることで、ビューンと移動して少し落下してまた動き出すという
実際のクラゲのかわいい動き方に近づけました。
海の中ではクラゲも天敵がいるので、それらを作りました。海藻、海洋ごみ、亀、マンボウの４種類作りました。
<ul style="list-style-type: none; padding-left: 0; margin: 0;">
<li>海藻 : 水の動きに翻弄されて左右にゆらゆらしています。</li>
<li>海洋ごみ : ごみなので海を漂っています。</li>
<li>亀 : 同じ方向をだらだらと泳いでいます。</li>
<li>マンボウ : 積極的にクラゲを食べようと追ってきます。</li>
</ul>
イラストは生成AIを活用して、作りました。
